# 课程简介
本课程主要讲解Microsoft Blend For Visual Studio 2019，帮你一步步掌握Microsoft Blend的那令人激动的超强功能。
<br/>
如果你正在使用WPF、UWP技术进行开发，你一定需要学习Microsoft Blend.
<br/>
在这里你会发现，只需动动鼠标，你就可以迅速开发出又酷、又炫、又华丽应用程序界面，而系统自动为你生成全部的XAML代码。从此告别手工编写XAML的苦逼日子。
